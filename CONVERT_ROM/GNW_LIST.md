There are only 56 unique games listed above. The following games have bug fixes or minor changes:

 * CN-07 Game & Watch: Helmet (Rev.1 / CN-07 original) as a 2 players on screen glitch when the unit is in time mode. It was replaced with CN-17, which is still labelled as a CN-07 on the device.
 * IP-05 Game & Watch: Judge (Green / Original) has a bug where one player would only earn one point for dodging, while the other would still gain two. IP-15 Game & Watch: Judge (Purple / Revised) was released at the same time which doesn't have this bug. Note: IP-15 still labelled as a IP-05 on the device.
 * CM-72A Mario's Cement Factory (Table Top, CM-72A) has a different start up melody to CM-72, otherwise they are identical.
